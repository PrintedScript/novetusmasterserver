Novetus Master Server

This server is being used as a Master Server which keeps tracks of servers running on Novetus. Please make sure you use a Novetus client to join any of these servers, you can download the latest client from here

I only plan to host this server for a few months, do not expect it to last forever.


If you find any problems or a malicious server please contact SomethingElse#0024 on Discord, if you are having problems with using the Novetus client please ask for help in the Novetus Discord server.

How to connect to master server

Client

On the Novetus client you will find a button that says Browse Servers, click on that button. A new window should pop up and in the Master Server Address paste in query.printedweb.me and click on the refresh button to see the server list.

Before joining it is also recommended that you generate a new user id and change the default username, using the default username may cause problems when joining the server.

Note: Not all servers on the master server list is hosted by me, it is also not recommended to join any server which is using a 2007 or older client as it may contain security risks that could damage your device.

Server

Before you start a server on the Novetus Client, you would find a box named Browser Master Server. If you want your server to show up on the server list you would have to paste in query.printedweb.me and your all ready to go! You can start your server and other people would be able to find your server.

Note: Putting your server on a master server list can be dangerous, if you only plan on using your server to play with friends please do not connect it to a master server list as it would allow other strangers to connect to your server and DDoS you. I am not responsible for any damage caused if your server does get DDoSed.
